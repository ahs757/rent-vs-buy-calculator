# 测试清单（手工验收）

## 1. 基础功能

1. 打开 `index.html`，页面正常渲染，无报错。
2. 修改任意输入值，结果区域应实时更新。
3. 拖动滑块时，对应输入框同步变化；反向输入也会同步滑块。

## 2. 输入校验

1. 清空任意输入框，显示“这里还空着呢...”提示。
2. 输入负数（非房价涨幅），显示“金额不能为负哦...”提示。
3. `房价年涨幅预期` 输入 `-3`，应允许并正常计算。
4. 首付比例、贷款年限、持有年限、房价涨幅超范围时，显示范围提示。

## 3. 结果正确性（冒烟）

1. 默认参数下应得到一组可读结果（万元 + 元/月）。
2. 将持有年限大幅增加（如 20 年），观察盈亏平衡点文案变化。
3. 将房价涨幅设为负值（如 -2%），建议文案应提示观望风险。

## 4. 交互体验

1. 在输入框按 `Enter`，焦点跳到下一个输入项。
2. 刷新页面后，输入值应从本地存储恢复。
3. 点击底部“关于我们/隐私政策”可正常打开页面。

## 5. 响应式

1. 桌面宽屏下卡片布局正常，内容不重叠。
2. 窄屏（手机宽度）下输入项和结果卡片可读、无横向滚动。
