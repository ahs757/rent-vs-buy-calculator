# 租房 vs 买房划算计算器（有温度版）

一个纯前端静态项目，用温暖文案 + 简化财务模型帮助用户判断“租房还是买房”。

## 功能清单

- 10 项输入参数，支持输入框与滑块联动
- 实时计算（`oninput`）无需点击按钮
- 输出买房总成本、租房总成本、月均成本、盈亏平衡点
- 等额本息月供与剩余本金计算
- 动态建议文案（首付比例、月供/租金比、房价预期等）
- 成本对比条形可视化
- 输入校验（空值/负数/范围）
- 本地存储（自动恢复上次输入）
- 适配桌面和移动端

也可以使用任意静态服务器打开项目目录。

## 关键计算说明（简化模型）

- 贷款总额 = 房价 × (1 - 首付比例)
- 月供 = 等额本息
- 卖出价 = 房价 × (1 + 房价年涨幅)^持有年限
- 买房净成本 = 首付 + 持有期已还月供 + 税费 + 物业费 + 维护费 - (卖出价 - 剩余贷款)
- 租房总成本 = 按年涨幅累加租金

备注：用于决策参考，不构成专业投资建议。

## GitHub Pages 部署

1. 新建 GitHub 仓库并推送当前目录文件。
2. 打开仓库 `Settings` -> `Pages`。
3. `Build and deployment` 选择 `Deploy from a branch`。
4. Branch 选 `main`（或你的默认分支），目录选 `/ (root)`。
5. 保存后等待部署完成，访问生成的网址。
